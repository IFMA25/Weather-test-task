# Cloudora Weather App

Cloudora — адаптивний застосунок погоди, розроблений як тестове завдання для Panda Team. Проєкт реалізований на Vue 3 та TypeScript із використанням OpenWeather API відповідно до вимог ТЗ.

## Можливості

Застосунок підтримує такий функціонал:

- OpenWeather API для отримання даних про погоду;
- пошук міста з автодоповненням;
- відображення погоди у вигляді картки на поточний день;
- графік температури на день і перемикання на прогноз на 5 днів;
- створення кількох карток погоди для різних міст, максимум до 5;
- видалення карток через модальне підтвердження;
- вкладка «Обране» з обмеженням до 5 міст і збереженням у localStorage;
- сповіщення про обмеження до 5 елементів за допомогою модального вікна;
- адаптивний інтерфейс для desktop і mobile;
- багатомовність: українська та англійська;
- прелоадери для станів завантаження;
- визначення погоди користувача за поточною локацією як стартового сценарію за допомогою ipapi.co;
- світлу та темну тему;
- перемикач теми з збереженням вибору в localStorage;

Окрім базових вимог, у проєкті також додано:

- роутінг Vue Router;
- сторінка 404;
- підключені тост-сповіщення (vue-sonner);


## Технології

- .env для змінних середовища;
- SCSS для стилів;
- Vue 3 для фронтенду;
- TypeScript для типів;
- Pinia для управління станом;
- Vue Router для маршрутизації;
- localStorage для збереження даних;
- JS chart plugin без Vue wrapper для побудови графіків;
- ESLint та Husky для лінтування коду;

## Запуск проєкту

### Встановлення залежностей

```bash
pnpm install
```

### Запуск у dev-режимі

```bash
pnpm run dev
```

### Продакшн-збірка

```bash
pnpm run build
```

## Змінні середовища

Для роботи з API потрібен ключ OpenWeather:

```env
VITE_OPENWEATHER_API_KEY=your_api_key_here
```
